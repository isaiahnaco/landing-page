# landing-page
A repository for my second project with The Odin Project

In this project, I am tasked with creating a landing page from scratch. I've been given a design to follow and the colors and fonts to use. To create the landing page, I will need to use what I've learned from practicing basic HTML and CSS concepts. These concepts include:

* HTML 
  * Elements
  * Tags
  * Lists
  * Images
* CSS
  * Flexbox
    * Axes
    * Alignments
